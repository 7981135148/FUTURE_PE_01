## 🧠 Prompt Logic & Framework

Each prompt was engineered with a specific structure to ensure high-quality, non-generic output:

### 1. Role Assignment
Every prompt begins by assigning a professional role to the AI:
```
"You are a professional website copywriter specializing in local fitness businesses."
```
This sets tone, expertise level, and output quality expectations.

### 2. Business Context Block
All real business details are passed upfront — name, location, phone, rating, audience. This ensures the AI generates **specific** copy, not generic filler.

### 3. Section-by-Section Instructions
Each section has clear sub-instructions with:
- What to write (headline, body, button text)
- Word/sentence limits
- Specific details to include (numbers, location names)
- Power words or style references

### 4. Tone & Rules Block
Every prompt ends with explicit rules:
- Avoid vague words ("amazing", "great")
- Use specific fitness language
- Write as if presenting to a real paying client
- No repeated phrases across sections

### 5. Output Format Instruction
```
"Output each section with a clear label."
```
This keeps the AI output organized and directly usable.

## 🔁 Reusability

These prompts are **fully reusable** for any local gym or fitness business. To adapt:

1. Replace business name, location, phone in the context block
2. Update the rating and member count
3. Adjust service names if different
4. Change tone from "energetic" to match the new brand

The same framework works for: salons, clinics, cafes, coaching centers, and agencies.
